EZ Payroll (Laravel)

A working online payroll system built using Laravel Framework
(Coding Bootcamp Project)

EZ-Payroll is a online payroll system designed for basic payroll computations. Features includes mass insertion/update of employee records, easy payslip generation and many more.

Objectives

To learn Laravel Framework and create a website using Laravel in which you need to apply all the things you've learned with all the previous projects given a time period of three weeks.

Challenges

Laravel was introduced to us two weeks before the deadline, meaning we already have written some of our codes during the first week. The challenge? First, I need to convert my codes from normal PHP to something that will utilize Laravel. Second, I need to learn Laravel two-weeks before the deadline which is not easy but worth it :D.

P.S. I haven't figured out yet how to do mail:queues in a production setting. Somebody help me.

Results

After three stressful weeks, I somehow manage to at least finish a one way trip to payslip generation. What does it mean? It means that everything will work fine as long as you use the website correctly or at least you're able to learn it correctly(Sorry there's no written documentation yet on how to use it). I admit that this payroll website have more loopholes than any of my previous projects. But overall I'm pretty happy with the result of this project, I learned so much and that's what important. I even learned basic bootstrap along the way and I feel like I don't want to do hard coded css again.

  • Creation Time: 3 weeks
  • HTML/CSS Experience Level: Proficient
  • PHP Experience Level: Proficient
  • MySQL Experience Level: Proficient
  • Laravel Experience Level: Beginner
  • Code Repository: https://github.com/devr52/EZPayroll

  • There might be further improvements to be implemented in the future.But for now it only serves as learning material.
